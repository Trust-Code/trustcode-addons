Project Task Search Custom

Search tasks for description and messages

The module allows to search a tasks for description and messages it contains in addition to task's name. To use go to
Project → Tasks menu and enter the text you want to search for in the search field.

For example, we enter "document" in the search field. As a result we can see 4 tasks

1. Two tasks are displayed, because "document" is contained in their names

2. One task is displayed, because it contains "document" in the description

3. Another one is displayed, because it contains "document" in its messages
